index.html is the main page and to access its full content open the file with VScode 
				or any other text editor and then open with a live server.

To open with live server, 

* IN VSCODE: Go to Extensions and download and install the extension for live server.
		Once installed, right click the code in index.html and click on "Open with Live Server" 
		to view the complete content.

IN CASE YOU COULDN'T INSTALL LIVE SERVER EXTENSION:
		The navigation bar will be not be visible as it is loaded from nav.html using jquery.
